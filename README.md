# pg18_ubi10
PostgreSQL 18 - Rocky 10 Ubi Micro base image

# Usage

⚠️You must mount /usr/share/zoneinfo, it doesn't exist in ubi-micro ⚠️

Ex :

```
docker run -d 
```
